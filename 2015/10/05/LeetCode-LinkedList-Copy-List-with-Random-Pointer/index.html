LeetCode-LinkedList: Copy List with Random Pointer

  • LeetCode

题干如下:

A linked list is given such that each node contains an additional random pointer which could point to any node in the list or null.

Return a deep copy of the list.



说实话，看到题目没什么思路，也不懂什么叫 deep copy 。

参考了网上的题目，这道题要求的其实是能够创建一条完整的linkedlist,然后这条链表中所有的nodes跟next指针都是新创建出来的，也就是说新的链表与旧的没有重叠(感觉有点过度解释了….)。

很容易想到就是两个指针，一新一旧copy就行了，但问题是一旦存在random指针后，random可能指向一个尚未出现在链表中靠后的node，那么如果我们直接用原链表node的值新建node后，怎么保证之后继续做链表复制时其他存在next指向该node的指针？

为了确保这一点，HashMap就派上用场了, 我们可以把 Pair中的Key设为原链表中的node，Value设为新链表中对应该node的复制node，一遍traversal之后，我们就有了复制好的除random指针外的新链表 。

之后再进行一遍traversal，针对每个原node1的random指针指向的node2, 我们通过HashMap中与node2匹配的新链表中node3, 然后让新链表中当前指针指向node3就行了 。
